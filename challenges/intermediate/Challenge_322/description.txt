# Description

Write a program that, given an integer input n, prints
the largest integer that is a palindrome and has two
factors both of string length n.

# Input Description

An integer

# Output Description

The largest integer palindrome who has factors each
with string length of the input.

# Sample Input:

[code]

    1
    
    2
    
[/code]

# Sample Output:

[code]

    9
    
    9009
    
[/code]

(9 has factors 3 and 3. 9009 has factors 99 and 91)

# Challenge inputs/outputs

[code]

    3 => 906609
    
    4 => 99000099
    
    5 => 9966006699
    
    6 => ?
    
[/code]
