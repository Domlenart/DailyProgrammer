# Description

Write a program that outputs the first recurring
character in a string.

# Formal Inputs & Outputs

## Input Description

A string of alphabetical characters. Example:

[code]

    ABCDEBC
    
[/code]

## Output description

The first recurring character from the input. From the
above example:

[code]

    B
    
[/code]

# Challenge Input

[code]

    IKEUNFUVFV
    PXLJOUDJVZGQHLBHGXIW
    *l1J?)yn%R[}9~1"=k7]9;0[$
    
[/code]

# Bonus

Return the index (0 or 1 based, but please specify)
where the original character is found in the string.
